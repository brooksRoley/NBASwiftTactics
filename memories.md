# AI Agent Memories & Context

- **App Name**: Basketball Tactics
- **Monetization**: Target is a $0.99 upfront cost on the iOS App Store.
- **Core Value Proposition**: Educational NBA-level schematic puzzles (teaching concepts like Drop, ICE, Switch All, Blitz) and an interactive tactics board for coaches/fans.
- **IP Strategy (Crucial)**: We deliberately scrubbed the Balldontlie API real data (Lakers, LeBron) and replaced them with mock archetypes ("L.A. Stars", "Generational Shooter") to ensure Apple does not reject the app for unauthorized commercial use of NBA intellectual property.
- **Current State**: We have 10 fully implemented puzzles. The app architecture uses MVVM in SwiftUI. The main views are `PuzzleView`, `BasketballCourtView`, and `RosterView`.
- **Next Milestone**: V1.0.0 App Store Submission.
